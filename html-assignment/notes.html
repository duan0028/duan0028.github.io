HTML Tags

a
<a> defines a hyperlink, which is used to link from one page to another.
img
<img> is used to Embed an image in an HTML page.
abbr
<abbr> represents an abbreviation or acronym.
em
<em> marks text that has stress emphasis.
ul
<ul> represents an unordered list of items, typically rendered as a bulleted list.
